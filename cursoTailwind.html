Instalacion Inicial

instalamos la extension a visual studio code pero que nos ayude a autocompletar se llama Tailwind CSS IntelliSense
tailwind css IntelliSense
creamos una carpeta ejemplo-tailwind e iniicializamos un proyecto de javascript : 
      
        npm init
instalamos tailwindCss client : 
      
        npm install tailwindcss @tailwindcss/cli
creamos una subcarpeta llamada src y detro de esta vamos a crear index.html y input.css
el codigo que vamos a colocar dentro del input.css es : 
      
        @import "tailwindcss";
modificamos nuestro package.json en la parte de escript con la siguiente linea: para no escribir simplemente npm run tailwind 
      
        
        "scripts": {
          "test": "echo \"Error: no test specified\" && exit 1",
          "tailwind": "npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch"
        },
NOTA:
nota por utlimo creamos dentro de la carpeta src un archivo Tailwind config.js en blanco, porque aveces no se actualiza dinamicamente los cambios.

Tipografia

Como agregar fuentes inter primero buscamos la fuentes en google font inter
en la pestaña de importar tomamos el link:
nuestro input.css quedaria de la siguiente forma: 
      
        @import url('https://fonts.googleapis.com/css2?family=Inter:ital,opsz,wght@0,14..32,100..900;1,14..32,100..900&display=swap');
        
        @import "tailwindcss";
        
        @theme{
            --font-inter: "Inter", sans-serif;
        }
el tamaño de la fuente tambien lo podemos cambiar por ejemplo la clase text-xs es el mas pequerño 0.75 rem o 12 pixelees, 
      
        <p class="text-xs">prueba de una frase</p>
        <p class="text-sm">prueba de una frase</p>
        <p class="text-base">prueba de una frase</p>
        <p class="text-lg">prueba de una frase</p>
        <p class="text-xl">prueba de una frase</p>
    
        <p class="text-2xl">prueba de una frase</p>
ahora el espaciado (interlineado) se utiliza /6 , /7 o /8 
      

        <p class="text-xs/6">prueba de una frase con varias lineas</p>
        <p class="text-xs/7">prueba de una frase con varias lineas</p>
        <p class="text-xs/8">prueba de una frase con varias lineas</p>
tambien podemos colocar las letra en cursiva, mayusculas, minusculas o capital: 
      
        <p class="italic">
            rodney zapata
        </p>
        <p class="not-italic">
            rodney zapata
        </p>
        <p class="uppercase">
            rodney zapata
        </p>
        <p class="lowercase">
            RODNEY ZAPATA
        </p>
        <p class="capitalize">
            rodney zapata
        </p>
el peso de la fuente se refiere es que tan grueso o que tan delgado es la fuente: 
      
        <p class="font-thin">
            peso de la fuente delgada
        </p>
        <p class="font-normal">
            peso de la fuente normal
        </p>
        <p class="font-bold">
            peso de la fuente bold
        </p>
        <p class="font-black">
            peso de la fuente negro
        </p>
        <p class="font-semibold">
            peso de la fuente semibold
        </p>
Tambien podemos personalizar el grosor por ejemplo creremos un grosor llamado ligero
el codigo en input.css quedaria de la siguiente forma: 
      
        @theme{
            --font-inter: "Inter", sans-serif;
            --font-weight-ligero: 12;
        }
y la forma de usarlo seria: 
      
       
        <p class="font-ligero">
            prueba con grosor ligero
        </p>
